🎓 Student Management System

A web-based Student Management System developed using ASP.NET MVC that allows users to efficiently manage student records including creation, update, deletion, and image uploads.

🚀 Features
➕ Add new student details
📝 Edit existing student information
❌ Delete student records
📋 View student list
🔍 Search functionality
🖼️ Upload and display student images
✅ Form validation
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: ASP.NET MVC (C#)
Database: SQL Server
